DOM QUESTIONS

1. What is the DOM tree?

DOM tree is a hierarchical structure of HTML elements (parent → child nodes).


2. Difference between textContent and innerHTML

textContent


• textContent Treats everything as plain text
• Ignores HTML tags (they show as text, not formatted)
• Safer (prevents script injection)


innerHTML


• Reads/sets real HTML
• Tags are interpreted and rendered
• Can be risky if used with user input


textContent= text only but innerHTML= text + HTML Structure


3. Difference between onclick and addEventListener

onclick → only one event, can be overwritten

addEventListener → multiple events, more flexible



EVENT QUESTIONS

1. What is an event in JavaScript?

An event is an action (click, input, submit) that happens in the browser.


2.What is an event handler?

An Event handler is a function that runs when an event occurs.


3. Difference between input and change events

input fires immediately while typing

change fires after user finishes (and leaves input)
